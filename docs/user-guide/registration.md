# Event Registration

## Public Registration
Each event has a unique URL formatted as:
`https://your-crm.com/register/[event-slug]`

### Features
- **Instant Validation**: Email and name required.
- **Capacity Checks**: Automatically closes if event is full.
- **Confirmation**: Sends an email with QR code upon success.

## Managing Registrations
Admins can view all registrations in the **Events** tab.
1. Click "Events" in the sidebar.
2. Select your event.
3. View the "Attendees" table.
