# Analytics & Reports

## Dashboard Overview
The main dashboard provides a high-level view of your CRM health.
- **Total Contacts**: Active database size.
- **Engagement Score**: Average engagement level calculated from opens/attendance.

## Exporting Data
Navigate to **Reports**.
- **Event PDF**: Download a beautifully formatted summary for stakeholders.
- **CSV Export**: Get raw data for Excel/Google Sheets analysis.

## Definitions
- **High Engagement**: Attended recent event AND opened last email.
- **Churn Risk**: Low engagement + no activity > 90 days.
